# Student Management System - MySQL

A relational database project built with MySQL to manage students, courses, attendance and marks.

## Tables
- Students
- Courses
- Attendance
- Marks

## Features
- Track student enrollment per course
- Calculate attendance percentage per student
- Analyze marks and find top performers
- Identify students below 75% attendance

## Tech Stack
- MySQL
- MySQL Workbench

## Key Queries
- Student wise average marks
- Attendance percentage calculation
- Top scoring students
- Course wise student count
